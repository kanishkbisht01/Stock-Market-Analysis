📊 NIFTY 50 Stock Market Analysis (2024–2025)

## Project Overview

This project presents a comprehensive analysis of companies listed in the NIFTY 50 index using financial and market data from 2024 and 2025.
The objective is to evaluate company performance, identify sectoral trends, analyze investor behavior, and generate actionable investment insights through a structured Excel dashboard.

## Objectives

* Evaluate company performance using key growth indicators
* Identify top-performing sectors based on aggregated growth metrics
* Analyze changes in shareholding patterns (FII, DII, Public, Promoters)
* Assess risk, valuation, and financial strength of companies
* Develop a decision-based framework for investment signals (Buy / Hold / Avoid)

##  Dataset Description

The dataset consists of NIFTY 50 companies with the following variables:

* Company Name
* Sector Classification
* Revenue (2024 & 2025)
* Profit (2024 & 2025)
* Stock Price (2024 & 2025)
* Shareholding Data (FII, DII, Public, Promoters)
* Financial Ratios (ROE, P/E Ratio, Beta)

##  Data Preparation

* Cleaned and structured raw data (removed duplicates, handled missing values)
* Standardized percentage formats for accurate calculations
* Organized dataset into multiple Excel sheets for analysis and dashboarding

##  Dashboard Features

An interactive Excel dashboard was developed with:

* Company selection dropdown (dynamic analysis)
* Auto-updating key metrics using lookup functions
* Growth comparison charts (Revenue, Profit, Price)
* Sector performance visualization (Top 10 sectors)
* Shareholding change analysis (FII/DII/Public/Promoters)
* Risk vs Return visualization
* KPI cards for quick insights

##  Key Findings

* Companies with strong profit growth generally showed positive price momentum
* Certain sectors consistently outperformed others, indicating sectoral strength
* Increased FII/DII participation often aligned with higher growth potential
* Overvalued stocks with weak fundamentals showed higher downside risk


##  Conclusion

This project demonstrates how raw financial data can be transformed into meaningful insights using Excel.
By combining growth metrics, sector analysis, and investor behavior, the dashboard provides a structured approach to stock evaluation and decision-making.
